## 📌 Project Overview

This project simulates the role of a **Junior Data Analyst** working with a public health agency to analyze global COVID-19 performance using **Microsoft Excel**.

The dashboard transforms raw pandemic data into **actionable insights** through advanced Excel functions, conditional formatting, and interactive visualizations.

🎯 Focus Period: **2020 – 2021 Global Pandemic Trends**

---

## 🎯 Objectives

✔ Clean and structure real-world dataset  
✔ Build analytical KPIs using Excel formulas  
✔ Highlight patterns using conditional formatting  
✔ Design an interactive analytical dashboard  
✔ Support data-driven decision making

---

## 📊 Dataset Information

| Column | Description |
|---|---|
| 🌎 Country | Nation Name |
| 🦠 Confirmed | Total Cases |
| ⚰️ Deaths | Fatalities |
| 💚 Recovered | Recovery Count |
| 📉 Active | Active Cases |
| 👥 Population | Country Population |
| 🌐 Continent | Geographic Region |
| 📅 First Case | Pandemic Start Date |

## 🧠 Excel Skills Demonstrated

### 🔢 Advanced Functions
- IF / IFS Logic
- KPI Calculations
- Death Rate %
- Recovery Rate %
- Infection Rate %
- VLOOKUP & HLOOKUP

### 🎨 Conditional Formatting
- Risk-based color highlighting
- Recovery performance heatmap
- Data bars for infection intensity
- Icon indicators for risk classification

### 📈 Visual Analytics
- Column Charts
- Pie Charts
- Combo Charts
- Trend Analysis
- Interactive Filters & Slicers
